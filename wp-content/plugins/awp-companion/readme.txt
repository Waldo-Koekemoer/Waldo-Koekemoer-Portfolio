A WP Life Companion is a useful plugin for awpbusinesspress WordPress theme by awordpresslife.

== Description ==

This A WP Life Companion plugin is created for awordpreslife themes. 
The plugin add custom frontpage sections for awordpresslife themes and it will allow you full customizer settings for the theme front page sections. 
The plugin options are user-friendly and you can use this plugin for awpbusinesspress WordPress theme.
